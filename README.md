## Overview

This is my solution to the Blog-preview-card challenge on Frontend Mentor.
The goal of this project was to create a clean, responsive layout that perfectly matches the given design. Working on this challenge, I learned to use a strict, organized CSS design system.
This project is built with a mobile-first approach without the use of @media, ensuring the layout looks professional and scales correctly on screens of any size.
